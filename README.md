# 📘 AI产品经理学习助手

基于 RAG + LLM 的智能学习工具，专为想要入职 AI 产品经理的无基础人员设计。支持**问答机器人**、**自适应测验**、**Flashcard 抽认卡**、**模拟面试**、**场景分析**、**每日一题**等模块，帮你系统化掌握 AI 产品知识。

## ✨ 主要功能

- 🤖 **知识库问答**：基于自建知识库（Markdown 文档）的 RAG 对话，可追溯来源。
- 📝 **自适应测验**：开放式问答题，连续出题，自动评分 + 低分详细辅导。
- 🃏 **Flashcard 抽认卡**：一键生成 15 张卡片（简单/中等/困难），支持掌握状态追踪与重复练习。
- 🎤 **模拟面试**：AI 依次提问 3 道面试题，限时回答，最后给出综合评分与改进建议。
- 🧩 **场景模拟**：模糊业务场景分析题，要求设计框架与验证方案，提供 5W2H 分析表格。
- 📅 **每日一题**：根据掌握度最低的知识点自动出题，记录历史，形成学习闭环。
- 📚 **知识库浏览器**：独立界面浏览、搜索、预览全部知识文件，不污染对话历史。
- 📊 **学习进度看板**：追踪每个知识文件的掌握度（正确率），定位薄弱环节。
